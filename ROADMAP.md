# Roadmap — ActuaryLab

## Objetivo

O ActuaryLab é um laboratório de estudos voltado para Ciência Atuarial, Estatística, Ciência de Dados, Gestão de Riscos e Finanças Quantitativas.

O projeto será desenvolvido de forma incremental, utilizando uma metodologia baseada em Sprints. Cada Sprint representa uma etapa de aprendizado, desenvolvimento de projetos práticos e consolidação de conhecimentos.

## Objetivos do Projeto

- Desenvolver conhecimentos em Ciência Atuarial.
- Aplicar Python na resolução de problemas atuariais.
- Aprender Estatística e Probabilidade através de projetos.
- Construir modelos de risco.
- Desenvolver dashboards para análise de indicadores.
- Construir um portfólio profissional.

---

## Metodologia

Cada Sprint será composta por:

- Estudo teórico
- Exercícios
- Projeto prático
- Registro da evolução (Learning Journal)
- Revisão da Sprint
- Publicação no GitHub

---

## Resultado Esperado

Ao final do projeto o ActuaryLab será um laboratório completo de estudos em Ciência Atuarial contendo documentação, código, projetos, notebooks e modelos matemáticos desenvolvidos ao longo do processo de aprendizagem.
