# ActuaryLab

> Laboratório de estudos em Ciência Atuarial, Ciência de Dados e Gestão de Riscos.

---

## Sobre o Projeto

O ActuaryLab é um projeto de longo prazo criado para consolidar conhecimentos em Ciência Atuarial por meio de projetos práticos.

O objetivo não é apenas estudar teoria, mas desenvolver aplicações utilizando Python, SQL, Estatística, Matemática Atuarial, Modelagem de Riscos e Ciência de Dados.

Todo o desenvolvimento será realizado utilizando uma metodologia baseada em Sprints, permitindo acompanhar a evolução do conhecimento ao longo do tempo.

---

## Objetivos

- Aprender Ciência Atuarial na prática.
- Aplicar Estatística e Probabilidade em problemas reais.
- Desenvolver projetos utilizando Python.
- Construir modelos de precificação de seguros.
- Simular reservas atuariais.
- Desenvolver dashboards.
- Construir um portfólio profissional.

## Roadmap

O desenvolvimento será dividido em Sprints.

Cada Sprint conterá:

- Conteúdo teórico
- Exercícios
- Projeto prático
- Documentação
- Registro de evolução

Consulte o arquivo **ROADMAP.md** para acompanhar todas as etapas.

---

## Metodologia

O projeto segue uma abordagem baseada em aprendizagem por projetos (*Project-Based Learning*), em que cada conceito estudado é imediatamente aplicado na construção de soluções práticas.

O fluxo de desenvolvimento será:

```
Estudar
      ↓
Aplicar
      ↓
Documentar
      ↓
Publicar
      ↓
Evoluir
```
